# SPHirit

Simulateur de mécanique des fluides 2D basé sur la méthode SPH (Smoothed Particle Hydrodynamics) avec formulation ALE (Arbitrary Lagrangian-Eulerian).

Développé dans le cadre d'un projet académique au CESI, en collaboration avec des enseignants-chercheurs.

## Cas tests disponibles

- **DamBreak** — rupture de barrage
- **Funnel** — écoulement visqueux dans un entonnoir (ISO 2431)
- **Jet** — impact de jets
- **TGV** — Taylor-Green Vortex
- **Car** — interaction fluide-structure
- **BodyF** — corps flottant
